How to use the iTunes File Sharing

1. Connect your iOS device to your computer.

2. In iTunes, select your iOS device, and then select apps menu.
3. Scroll down to bottom and Find file sharing.

4. Below file sharing, select “intoLive” from the list.
5. Your gif, mp4, mov files drag & drop to “intoLive Documents”

6. Launch intoLive and Select itunes menu.
7. You can see yours gif and video files.
