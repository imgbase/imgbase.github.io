2. 잠금화면에서 라이브포토가 움직이지 않습니다.

잠금화면 라이브포토 기능은 6s 이후에 출시된 기기에서만 사용이 가능합니다. 해당 기능은 3D 터치가 가능한 기기에서만 사용하실 수 있으며, 애플은 그 이전 기기에 대해 해당 기능을 만들지 않았기 때문에 잠금화면에서 라이브포토를 설정하실 수가 없습니다.

지원되는 기기 : 6s, 6s Plus, 7, 7 Plus, 8, 8 Plus, X
지원되지 않는 기기 : 5SE, 5s, 6, 6 Plus, iPad, etc.

3. iTunes 파일 공유 사용방법

1. 기기를 PC에 연결하고 PC에서 iTunes 를 실행합니다.

2. 앱 메뉴를 선택하고 스크롤을 아래로 내려 iTunes 파일 공유로 이동합니다.

3. iTunes 파일 공유에서 intoLive 앱을 선택합니다.
4. GIF/비디오 파일을 드래그 & 드랍 또는 추가 버튼을 이용하여 파일을 추가합니다.

4. 잠금화면에 라이브포토를 설정하면 소리가 들리지 않습니다.

잠금화면에서는 소리의 재생을 허용하지 않습니다. 해당 기능은 기기 자체에서 제공하지 않는 기능으로 저희도 언젠가 Apple이 제공해주길 기대하고 있습니다.

5. 페이스북 페이지에 라이브포토를 업로드하는 방법

페이브북 페이지에 라이브포토를 업로드할 때는 Page 앱(주황색 아이콘)이 아닌 Facebook 앱(파란색 아이콘)을 사용하셔야 합니다. 아직 Page 앱은 라이브포토 업로드를 지원하지 않습니다.

6. 라이브포토를 재생하면 확장되어 추가한 글자가 잘립니다.

라이브포토 형식은 재생될 때 영역을 약 8% 정도 확장해서 재생합니다. 라이브포토에 텍스트를 추가하는 경우, 원본 비디오 또는 사진의 가장자리에서 약 10% 떨어진 위치에 텍스트를 배치하는 것이 좋습니다. 비디오의 가로 폭이 1000px 인 경우 가장자리에서 100px 떨어진 곳에 텍스트를 추가하세요.

7. 안드로이드 버전은 왜 없습니까?

라이브포토 파일 형식은 iOS의 고유한 형식으로 안드로이드에는 존재하지 않기 때문에 안드로이드용 intoLive 앱은 지원할 계획이 없습니다.

8. 구매를 했는데 프로버전을 사용할 수 없어요.

구매페이지에서 구매복구 버튼을 다음과 같이 눌러주세요.

1. 앱스토어 로그아웃
2. 기기의 전원을 종료
3. 앱스토어 로그인
4. ImgPlay 앱 구매페이지의 구매복구 버튼 누르기



로그인 로그아웃을 몇번 시도하고도 복구가 되지 않는다면, 앱스토어에서 구매한 영수증을 다시 한번 확인해주세요.

9. 중복결제가 된 것 같아요. 결제오류가 있어요.

결제시스템은 전부 앱스토어에 의해 관리가 됩니다. 만약 지불된 금액에 문제가 있다면 다음 링크를 클릭하고, 상세 영수증을 다시 확인해보세요. 문제가 있다면 앱스토어에 해당 구매 건에 대하여 문의를 남길 수 있습니다.

10. 구독을 취소해주세요.

intoLive 앱은 매월 결제되는 구독형 가입모델이 아닙니다.
한번 결제로 영원히 프로버전 앱을 사용하실 수 있습니다. 걱정하지 말고 사용하세요~!😃
