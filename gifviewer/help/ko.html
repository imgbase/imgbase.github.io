GIF Viewer

1. 움짤(GIF)이 재생이 안되요!

아이폰/아이패드에서 GIF를 다운로드 할 때, GIF처럼 보이지만, 움직이는 애니매이션 GIF가 아닌 정지된 사진이미지를 저장하는 경우가 있습니다.
GIF를 저장할 때는 Safari 앱을 이용하시는 것을 추천드립니다.

Safari 앱이 아닌 기타 앱(Naver, Daum 등)을 이용하여 저장하면 GIF가 아닌 정지된 사진 이미지를 저장하는 경우가 있습니다. 다른 앱들이 GIF 파일 포맷을 지원하지 않아서 생기는 문제로 최근 iOS에서 GIF를 지원하는 앱들이 늘어나고 있지만 아직은 지원하지 않는 곳이 더 많습니다.

2. GIF가 갑자기 보이지 않아요. (iOS 8 사용자에게만 해당)

GIF가 iCloud 사진보관함에 자동으로 업로드되었을 가능성이 큽니다.

iCloud 사진 보관함을 "iPhone 저장 공간 최적화"로 사용하면 자동으로 기기에 저장된 사진을 삭제하고 클라우드에 저장합니다. 그로 인해 클라우드에 저장된 GIF를 불러올 수가 없습니다. GIF Viewer 앱은 기기에 저장되어 있는 GIF만 재생할 수 있습니다.

(iOS 9이상에서는 iCloud 에 업로드된 GIF도 정상적으로 가져옵니다.)

다음과 같은 2가지 방법 중 하나를 사용하세요.

방법 1. iCloud 사진 보관함 설정에서 "다운로드 및 원본 유지"

설정 앱 -> 사진 및 카메라 -> iCloud 사진 보관함에서 "다운로드 및 원본 유지"를 선택하세요.

* 이미 클라우드에 저장된 파일은 다시 기기에 저장하면 제대로 볼 수 있습니다.

방법 2. GIF Viewer 앱 안의 앨범으로 GIF를 복사

GIF Viewer 앱 안의 앨범으로 복사한 GIF 파일은 클라우드의 영향을 받지 않습니다. iCloud 사진 보관함을 "iPhone 저장 공간 최적화"로 사용하고 싶다면 GIF 파일이 클라우드에 올라가기 전에 GIF Viewer 앱의 앨범에 복사하세요.

3. iTunes 파일 공유 사용 방법

1. 아이폰 또는 아이패드를 컴퓨터에 연결하고 컴퓨터에서 iTunes를 실행하세요.

2. 앱 메뉴를 선택하고 스크롤을 아래로 내려 iTunes 파일 공유로 이동 하세요.

3. iTunes 파일공유에서 “GIF Viewer”를 선택 하세요.

4. GIFViewer앱 안으로 이동하고 싶은 GIF파일들을 드래그하여 그림과 같이 GIF Viewer Documents 추가 하세요. (폴더를 이동하면 앱에서 GIF를 인식하지 않습니다. GIF파일만 넣어 주세요)

5. 아이폰, 아이패드에서 GIF Viewer를 실행 시킨후 “iTunes 파일 공유” 메뉴를 선택하면 추가한 GIF들을 볼 수 있습니다.

사용시 궁금한 점이 있으면 cs@imgbase.me 으로 메일을 보내주세요.

4. 트위터로 공유하거나 다운받을 때 문제가 발생해요.

아이폰 일반설정에서 트위터 계정 설정이 잘 되어 있는지 확인하세요.

1. 설정 앱을 실행하세요.



2. 트위터를 찾아서 클릭하고 상세보기로 들어가세요.



3. 이 곳에 트위터 계정설정이 잘 되어 있는지 확인하고 계정이 없다면 계정을 추가하세요.
문제가 지속적으로 발생하면 이 곳의 계정을 전부 삭제한 후 다시 등록하세요. 가끔 트위터에서 계정연동의 오류가 발생합니다.
하단에서 GIF Viewer 앱 접근이 수락되어있는지도 확인하세요.



5. 트위터에서 움짤(GIF)을 다운받는 방법

1. 트위터에서 움짤(GIF)이 존재하는 트윗에서 “...” 을 클릭하세요.
2. “공유하기”를 선택하세요.
3. “트윗 링크 복사하기”를 선택하세요.
4. GIF Viewer 앱을 실행해서 “+” 버튼을 클릭하세요.
5. "Twitter GIF" 를 선택하면 움짤(GIF)을 다운받을 수 있습니다.

6. 웨이보로 공유할 때 문제가 발생해요.

아이폰 일반설정에서 웨이보 계정 설정이 잘 되어 있는지 확인하세요.

1. 설정 앱을 실행하세요.



2. Sina Weibo를 찾아서 클릭하고 상세보기로 들어가세요.



3. 이 곳에 웨이보 계정설정이 잘 되어 있는지 확인하고 계정이 없다면 계정을 추가하세요.
문제가 지속적으로 발생하면 이 곳의 계정을 전부 삭제한 후 다시 등록하세요. 가끔 웨이보에서 계정연동의 오류가 발생합니다.
하단에서 GIF Viewer 앱 접근이 수락되어있는지도 확인하세요.



7. 백업하는 방법

드랍박스 와 구글드라이브, 그리고 iTunes 파일공유를 통해 백업을 할 수 있습니다. (iTunes 파일공유 백업 : 도움말 8번 참고)

드랍박스, 구글드라이브에 GIF를 내보내는 방법
1. 내보내고 싶은 GIF를 선택하세요.
2. 공유하기 > 내보내기에서 드랍박스 또는 구글드라이브를 선택하세요.
3. 내보내기가 완료되면 드랍박스 > “앱” 폴더 > “GIF Viewer” 폴더에서 내보내기한 파일들을 확인할 수 있습니다. (구글드라이브는 구글드라이브 > "GIF Viewer" 폴더에서 바로 확인가능)

8. PC 로 GIF 를 이동하는 방법 (iTunes 파일공유)

iTunes 파일공유에서 GIF Viewer 를 선택하세요.

폴더를 그대로 드래그하여 PC로 이동하세요.

백업이 완료되면, PC의 폴더에서 GIF와 함께 PNG 이미지가 같이 저장되어 있습니다. PNG 파일만 지우시면 됩니다. GIF Viewer 앱은 앱의 속도향상을 위해 썸네일(PNG)을 생성하여 가지고 있습니다. 파일 갯수가 2~3배로 늘어 불편하시겠지만, 이 부분은 양해를 부탁드립니다.
