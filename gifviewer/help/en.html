1. GIF is not animating?

When GIF is downloaded on iPhone/iPad, there are cases that a GIF is saved as a Non-animating JPEG image.
We recommend using Safari app when saving GIF files.

Unless using other Apps than Safari app, not GIF but stopped (no video file) images get saved from time to time. It is a systematic problem that occurs because other apps do not support GIF file format. Even though apps supporting GIF under iOS are increasing, there are still many apps which do not support GIF program.

2. GIF is suddenly not showing. (Only for iOS8 users)

If you use the iCloud Photo Library, can be a problem.

When using “Optimize iPhone Storage” iCloud Photo Library, it automatically deletes photos saved in the device and saves them into iCloud. Therefore GIF files cannot be retrieved from iCloud. GIF Viewer app helps you perform on the GIF files saved within the device only. (For iOS9 and higher, GIFs uploaded to iCloud can import normally.)

Choose one of these two methods and practice it.

Method 1. “Download and Keep Originals” under the setting of iCloud Photo Library

Settings App -> Photos & Camera > iCloud Photo Library, select “Download and Keep Originals”

* Files already saved in iCloud can be retrieved again into your device later.

Method 2. Copy GIF into the album within GIF Viewer app

GIF files copied into the album within GIF Viewer app do not get affected by iCloud. If you would like to use “Optimize iPhone Storage” iCloud Photo Library, copy the GIF files into the album of GIF Viewer App before uploading them into iCloud.

3. How to use itunes file sharing?

1. Connect your iOS device to your computer.

2. In iTunes, select your iOS device, and then select apps menu.
3. Scroll down to bottom and Find file sharing.

4. Below file sharing, select “GIF Viewer” from the list.
5. Your gif files drag & drop to “GIFViewer Documents”

6. Launch GIFViewer and Select iTunes File Sharing menu.
7. You can see yours gif files.

4. There are problems when sharing or downloading to Twitter.

Please You confirm Twitter account setting in iPhone general Settings app.

1. In Settings app.



2. Find Twitter icon and Click that.



3. Check if the Twitter account is properly set, and if there is no account, please add one. If the problem continues, delete every account here and register them again. Sometimes account sync error occurs with Twitter. Also, check if GIF Viewer app access is on at the lower part.



5. How to download the GIF on Twitter

1. Click the "..." from the GIF Tweets
2. Select the "Share via..."
3. Select "Copy link to Tweet."
4. Open the GIF Viewer app, please click the "+" button.
5. Select the "Twitter GIF". You can download a GIF on Twitter.

7. How to create Backups

You can create backups through Dropbox, Google Drive and iTunes File Sharing. (Backup through iTunes File Sharing: Check Help No. 8)

How to upload to Dropbox or Google Drive
1. Select the GIFs you want to upload.
2. Go to Share>Export and select Dropbox or Google Drive
3. You can see your files in Dropbox>"Apps" folder>"GIF Viewer" folder. (Google Drive > "GIF Viewer" folder for Google Drive)

8. How to move GIF to PC (iTunes File Share)

Select GIF Viewer at the file share of iTunes.

Drag your folder directly to the PC.

When back up is completed, GIF and PNG are both saved in the folder in PC. You only have to delete the PNG files. GIF Viewer creates and keeps thumbnails(PNG) for a better speed of the app. The increased number of files from two to three times may come as an inconvenience, we ask for your consent.
