1. A foto fica com uma resolução fraca quando guardada em GIF

O formato de ficheiro GIF tem um limite de 256 cores. Por este motivo, é impossível evitar que a fotografia seja danificada quando for guardada em formato GIF. Recomendamos guardar a imagem no formato de vídeo se quiser guardá-la com alta qualidade.

2. The saved GIFs are not animated.

Android version 4.4 or below may not show animated gif on default Gallery app. If a gif can't be animated, use the "QuickPic" app. It can download from Google Play.

3. Why is there no sound?

The ImgPlay app focuses on creating GIF files, a format which does not support audio, so there is no sound.

4. Frame limit / Why can't I make a longer GIF?

ImgPlay allows you to create a GIF max 100 ~ 150 frames.
Photos to GIF : 100 frames
Video to GIF : 100 ~ 150 frames

Mobile devices typically have smaller system memory totals than PCs, so a frame amount greater than 100 would risk crashing most mobile devices. Therefore there is a limit.

5. A tradução parece ter alguns erros

Estamos a tentar disponibilizar a melhor tradução possível. Se tiveres alguma dificuldade em entender algo por causa da tradução, não deixes de nos informar. Será editada quando for atualizada. Agradecemos muito a tua ajuda e apoio.
