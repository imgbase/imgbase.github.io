1. La foto pierde resolución cuando se guarda como GIF

Los archivos en formato GIF tienen un límite de 256 colores, por lo que no se puede evitar alterar la foto cuando se guarda como GIF. Recomendamos guardar una imagen en formato vídeo si se quiere preservar la calidad alta.

2. Los GIF guardados no son animados.

Es posible que la versión 4.4 de Android e inferiores no muestren los GIFs animados de forma predeterminada en la aplicación Galería. Si no se puede animar un GIF, utiliza la aplicación «QuickPic». Se puede descargar desde Google Play.

3. ¿Por qué no hay sonido?

La aplicación ImgPlay se centra en crear archivos GIF, un formato que no dispone de audio. Para mantener el sonido original, tienes que guardar en un formato de vídeo sin editar la velocidad o la dirección en Editar vista.

4. Límite de fotogramas / ¿Por qué no puedo hacer un GIF más largo?

ImgPlay allows you to create a GIF max 100 ~ 150 frames.
Photos to GIF : 100 frames
Video to GIF : 100 ~ 150 frames

Mobile devices typically have smaller system memory totals than PCs, so a frame amount greater than 100 would risk crashing most mobile devices. Therefore there is a limit.

5. Parece haber algunos errores de traducción

Procuramos ofrecer la mejor traducción posible. Si tienes problemas para entender algo debido a la traducción, dínoslo. Se modificará para la próxima actualización. Agradecemos enormemente tu ayuda.
