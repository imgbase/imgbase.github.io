1. У фотографии ухудшается качество при сохранении в GIF

У формата GIF есть ограничение в 256 цветов, поэтому при сохранении фотографии в формат GIF неизбежна потеря качества. Если вы хотите сохранить высокое качество, мы рекомендуем сохранить изображение в видеоформате.

2. The saved GIFs are not animated.

В устройствах на Android 4.4 или ниже анимированные GIF могут не показываться в галерее. Если GIF не анимированы, используйте приложение QuickPic. Его можно установить из Google Play.

3. Почему нет звука?

ImgPlay — это приложение для создания GIF-файлов, а этот формат не поддерживает аудио.

Чтобы сохранить исходный звук, сохраните файл в видеоформате, не изменяя скорость или направление воспроизведения в режиме редактирования.

4. Лимит кадров / Почему нельзя сделать GIF подольше?

ImgPlay allows you to create a GIF max 100 ~ 150 frames.
Photos to GIF : 100 frames
Video to GIF : 100 ~ 150 frames

Mobile devices typically have smaller system memory totals than PCs, so a frame amount greater than 100 would risk crashing most mobile devices. Therefore there is a limit.
