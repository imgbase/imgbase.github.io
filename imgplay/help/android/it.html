1. Quando la foto viene salvata in GIF, è in bassa risoluzione

Il file in formato GIF ha un limite di 256 colori, perciò è inevitabile che la foto salvata in GIF sia di qualità inferiore. Per ottenere una qualità superiore, suggeriamo di salvare l'immagine in formato video.

2. Le GIF salvate non sono animate.

La versione 4.4 o inferiore di Android potrebbe non mostrare GIF animate nell'app Galleria predefinita. Se non è possibile animare una GIF, usa l'app "QuickPic". Puoi scaricarla da Google Play.

3. Perché non c'è suono?

L'app ImgPlay si concentra sulla creazione di file GIF, un formato che non supporta l'audio.

Per mantenere l'audio originale, è necessario salvare in un formato video senza modificare la velocità o la direzione in Modifica visualizzazione.

4. Limite frame / Perché non posso creare una GIF più lunga?

ImgPlay allows you to create a GIF max 100 ~ 150 frames.
Photos to GIF : 100 frames
Video to GIF : 100 ~ 150 frames

Mobile devices typically have smaller system memory totals than PCs, so a frame amount greater than 100 would risk crashing most mobile devices. Therefore there is a limit.
