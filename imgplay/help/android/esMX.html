1. La foto tiene poca resolución cuando se guarda como GIF.

El formato de archivo GIF tiene un límite de 256 colores, así que hay un daño inevitable a la foto cuando se guarda en GIF. Recomendamos guardar una imagen en formato de video si quieres conservar una alta calidad.

2. The saved GIFs are not animated.

La versión 4.4 de Android o menor no puede mostrar GIF animados en la aplicación predefinida de Galería. Si no se puede animar un GIF, usa la aplicación "QuickPic". Se puede descargar en Google Play.

3. ¿Por qué no hay sonido?

La aplicación ImgPlay se centra en la creación de archivos GIF, un formato que no es compatible con el audio.

Para mantener el sonido original debes guardar en un formato de video sin editar la velocidad o dirección en la vista de Editar.

4. Límite de cuadros / ¿Por qué no puedo hacer un GIF más largo?

ImgPlay allows you to create a GIF max 100 ~ 150 frames.
Photos to GIF : 100 frames
Video to GIF : 100 ~ 150 frames

Mobile devices typically have smaller system memory totals than PCs, so a frame amount greater than 100 would risk crashing most mobile devices. Therefore there is a limit.
