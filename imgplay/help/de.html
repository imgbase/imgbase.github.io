1. Picture gets low resolutions when saved into GIF

GIF file format has limitation of 256 colors, so there are unavoidable damage to the picture when it is saved in GIF. We recommend saving an image in video format if you want to save a high quality.

2. Picture loading takes too long

If the picture saved in the machine is uploaded in iCloud, loading may take long. The download is through the network, so the process is vulnerable to the network speed.

Try again in Wifi network.

3. I saved a GIF in the picture album but it doesn't move.

iPhone's photos app doesn't support GIF, so it seems still image. To see a gif on an iPhone, you need to install a GIF album App.

We recommend out GIFViewer App. It's free.

4. I cannot restore my payments. What can I do?

Payment processing is managed by Apple in its entirety.
First, You try logging out from the App Store and restarting your device before trying again.

If still it has a problem, please contact Apple for help.
