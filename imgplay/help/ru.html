1. У фотографии ухудшается качество при сохранении в GIF

У формата GIF есть ограничение в 256 цветов, поэтому при сохранении фотографии в формат GIF неизбежна потеря качества. Если вы хотите сохранить высокое качество, мы рекомендуем сохранить изображение в видеоформате.

2. Загрузка фотографий идет слишком долго

Загрузка фотографий может быть долгой, если они сохранены в iCloud. Их загрузка происходит через Интернет, поэтому ее скорость зависит от скорости соединения.

Попробуйте еще раз через Wi-Fi соединение.

3. Анимированные GIF поддерживаются только iOS 11 или более поздними версиями

Только устройства с iOS 11 или более поздней версией могут показывать анимированные GIF в приложении для фотографий.

Если вы не используете iOS 11, вам нужно установить обновление. Если у вас по какой-то причине не получается обновиться до iOS 11, мы рекомендуем установить приложение GIF Viewer. Это бесплатное и удобное приложение, которое мы разработали для просмотра анимированных GIF в приложении для фотографий.

4. Почему нет звука?

ImgPlay — это приложение для создания GIF-файлов, а этот формат не поддерживает аудио.

Чтобы сохранить исходный звук, сохраните файл в видеоформате, не изменяя скорость или направление воспроизведения в режиме редактирования.

5. Лимит кадров / Почему нельзя сделать GIF подольше?

Максимальное число кадров зависит от устройства.

Макс. 100 кадров: 4, 4S, 5, 5S, 6, 6 Plus
Макс. 150 кадров: SE, 6S, 6S Plus, 7, 8, iPad
Макс. 200 кадров: 7 Plus, 8 Plus, X, iPad Pro

На мобильных устройствах обычно меньше системной памяти, чем на ПК, поэтому при числе кадров больше 150 большинство мобильных устройств могут не справиться с нагрузкой. Поэтому существует лимит.

6. Как отправлять GIF в Facebook

Facebook позволяет делиться только URL-адресами GIF-файлов. Если вы хотите загрузить GIF в Facebook, вам нужно создать веб-ссылку на GIF с помощью Tumblr, GIPHY, Google Drive, личного сервера и т. д.

Приложение ImgPlay не загружает GIF-файлы пользователей на сервер и не может создавать ссылки на них. Если вы хотите загружать созданные в ImgPlay материалы напрямую, сохраните их в видеоформате и загрузите в Facebook.

7. При редактировании gif отображается только один кадр

Откуда у вас этот gif?
Если вы скачали gif-файл с веб-страницы, он может быть неанимированным. Некоторые приложеня не поддерживают скачивание анимированных gif для iOS. Если пользователь скачивает анимированный gif с их помощью, он может быть автоматически сконвертирован в неподвижное изображение, и только выглядеть как анимированный gif. По этой причине ImgPlay может показывать только однокадровые gif-файлы.

Мы рекомендуем приложение Safari для скачивания gif из сети. Safari — приложение-браузер по умолчанию для iOS, которое может скачивать анимированные gif-файлы.
Попробуйте скачать анимированный gif с помощью Safari.

8. Я хочу стереть сохраненные gif-файлы из приложения.

Если вы хотите удалить gif, сделанные в ImgPlay, их можно удалить из приложения для фотографий по умолчанию.

ImgPlay может сохранять gif-файлы пользователя только в приложение для фото по умолчанию. Оно не сохраняет их нигде больше, включая само приложение. Найдите свои gif в приложении для фотографий и удалите их там.
