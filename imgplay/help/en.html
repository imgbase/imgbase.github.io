1. Picture gets low resolutions when saved into GIF

Image format used often in web or mobile are JPG, GIF, and PNG.
A motion picture is GIF.
GIF has limitation of 256 colors that can be saved in a picture, so we recommend saving an image that has more than 256 colors in video format if you want to save a high quality video, because there are unavoidable damage to the picture when it is saved in GIF

2. Picture loading takes too long

If the picture saved in the machine is uploaded in iCloud, loading may take long. The download is through the network, so the process is vulnerable to the network speed.
Try again in Wifi network.

3. I saved a GIF in the picture album but it doesn't move.

iPhone's standard picture album doesn't support GIF, so it seems still. To see a motion picture on an iPhone, you need to install a GIF album App.

We recommend out GIFViewer App. It's free.
