1. Original Image seems to be unable to be saved.

ImgClip can save the original copy of web page which currently viewed.

However, certain web pages have mobile version, separately from PC version. Certain web pages are thumbnail images on the mobile version. Therefore, when unable to save the original copy, please select PC mode to run ImgClip. PC mode button is often located at the bottom of web page.

2. The image, smaller than the screen, seems to be broken since it is stretched. Is there any way to see the image complete?

You can edit the setting to view the image as original size on Setting within App -> Image -> Viewer Mode.

3. Size of App seems to be too big. Can its size be reduced?

Cache data are being stored to increase the reaction time of the App.

If you delete cache data on the Setting within App -> Cache Data menu, the size of App will be reduced accordingly.

4. How can I backup the file?

Backup feature is not currently offered. Sooner than later, backup feature through Dropbox Synchronization will be developed. Please wait for a while.

5. There are certain web pages where I cannot collect images. Why is that?

We are optimizing the App on every update so that it allows to collect images on as many web pages as possible. However, certain web sites are forbidding any image collection to protect copyrights. In such case, there is no way for us to let anyone collect image.

When you bump into the web page where you cannot collect image normally, please send us an email with the details at cs@imgbase.me. After analyzing the page, we will review the system for optimizing at next update.
